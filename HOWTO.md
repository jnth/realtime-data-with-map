# Utilisation

Interface de consultation des données :

    GET /

Pour envoyer des données :

    GET /import?lon=5.67&lat=44.876&time=2017-11-02_12:34:56&value1=26&value2=115.6&value3=0.114

Pour nettoyer la base de données :

    GET /clean
